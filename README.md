# Virtual Friends 🌟

Virtual Friends is an entertainment platform where users can discover and chat with AI-generated fictional characters.

> ⚠️ Disclaimer:
> All profiles on this platform are AI-generated fictional characters created for entertainment purposes only. They are not real people.

## Features

- 🤖 AI Generated Characters
- 💬 Interactive Chat System
- 🎮 Gamification & Rewards
- 🏆 Achievements & Leaderboards
- 📱 Mobile Friendly Design
- 🌙 Dark Mode Support
- 🔗 Invite & Share System
- 📊 User Dashboard
- 💰 AdSense Ready Layout

## Character Categories

- Friendly
- Funny
- Gamer
- Student
- Traveler
- Entrepreneur

## Technology Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Firebase
- Authentication: Firebase Auth
- Database: Firestore
- Hosting: Firebase Hosting / Vercel

## Installation

```bash
git clone https://github.com/yourusername/virtual-friends.git

cd virtual-friends

npm install

npm start
```
